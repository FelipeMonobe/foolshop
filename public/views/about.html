JavaScript fullstack

  • MongoDB: document-based NoSQL database.
  • ExpressJS: back-end web application framework.
  • AngularJS: front-end MVC web application framework.
  • NodeJS: server-side runtime enviroment.

    Bower packages

  • Angular: front-end MVC web application framework.
  • Angular-animate: official Angular animation module.
  • Angular-motion: CSS animations as Angular directives.
  • Angular-route: official Angular routing module.
  • Angular-strap: bootstrap as Angular directives.
  • Bootstrap: CSS framework with components.
  • Bootstrap-additions: CSS extension kit for bootstrap.
  • jQuery: Javascript functions library.
  • Ng-file-upload: Angular directive for file upload handling.

    Node modules

  • Body-parser: middleware for http requests parsing.
  • Cookie-parser: middleware for cookie parsing.
  • Ejs: HTML view engine.
  • Express: back-end web application framework.
  • Express-load: module for script loading into application.
  • Express-session: middleware for session handling.
  • Gulp: Javascript task and build runner.
  • Gulp-concat: Script concatenation gulp plugin.
  • Gulp-jshint: Script syntax linting plugin.
  • Gulp-strip-debug: Script debugger removal plugin.
  • Gulp-uglify: Script minification plugin.
  • Mongoose: Database schema module.
  • Mongoose-auto-increment: Database field auto increment plugin.
  • Mongo-sanitize: Database query selector sanitizer module.
  • Multer: middleware for multipart/form-data handling.
